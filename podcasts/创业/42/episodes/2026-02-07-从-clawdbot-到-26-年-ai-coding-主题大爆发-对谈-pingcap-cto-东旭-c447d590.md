最近 Clawdbot（现在叫 OpenClaw）各种刷屏 🦞。事实上，不只是 Clawdbot，从 25 年年底开始，Claude Code、Cowork、Skills 等产品和概念就一直在霸占我们的信息流。相信很多朋友和我们一样，在应接不暇的同时，也产生了各种疑问：到底这些产品都是做什么的？为什么这些产品会在最近密集爆发？技术上出现了什么新范式？AI 的未来又会发生怎样的变化？为了得到答案，我们请回了一位老朋友——PingCAP CTO 东旭。在上一期节目中，东旭更多是以联创的身份，分享了他关于全球化的思考与方法论。但其实，他还有很多面相，比如野生音乐人、哲学爱好者，以及始终在技术最前沿冲浪、甚至每天能烧掉 billion 级别 token 的 builder。所以，他可能是这期节目最合适的嘉宾人选之一。在这期节目里，东旭就从大量一手经验出发，分享了他对各种热门产品的实操体悟，用简单好懂的方式讲清楚了许多技术变化，并给出了他对 AI 未来走向的判断与推演。在讨论过程中，我们也梳理出了一条 AI 的演进线索：AI 产品，正在从「套壳大模型」，向「套壳 Coding Agent」方向演进。未来，一切可能都建立在 Coding 和 Agent 之上。最后，不得不再一次感慨，AI 的变化实在太快了。新产品层出不穷，我们似乎也在不断逼近 AI 拥有自我演化能力，甚至是某种「终极形态」的临界点。在这样的节点上，或许我们不只需要高密度地输入信息，也需要面对随之而来的迷茫与焦虑。所以在节目结尾，我们还请东旭回答了几个彩蛋问题，希望也能给一些朋友带来一点平静和启发吧～【人类博物馆】导游：曲凯，42章经创始人35 号珍藏：黄东旭，PingCAP 联合创始人、CTO，野生音乐人【时光机】Part 1 实操体悟 2:04 AI Coding 已经跃过了「奇点时刻」 3:18 25 年，AI Coding 的能力从多少分提高到了多少分? 4:44 未来应该禁止人类写代码？ 6:49 Claude Code 开创了什么新范式？ 8:35 过去一年，模型与 Agent 大幅进步的两个关键指标 13:54 Cowork 被高估了？ 15:15 为什么说 Skills 比 MCP 更适合 Agent？ 19:06 简单讲讲 Clawdbot 到底是个啥 21:22 关于 Clawdbot 未来可能性的一个猜想 21:48 Clawdbot 在本地跑 vs 在云端跑，有什么不同？ 23:21 Clawdbot 和 Manus 有什么本质区别？ 25:40 到底能用 Clawdbot 来干啥？一些真实的 Usecases 27:59 神奇（可怕）：AI 已经能自我演化了？ 33:43 Clawdbot 最重要的意义Part 2 个人思考 35:14 今年 AI 可能会怎样发展？ 37:10 未来 AI Coding 和人，应该如何分工？ 40:52 创业可能只剩下两个方向 45:50 给工程师们的建议 48:21 AI 时代的组织形式会发生什么变化？ 50:07 如果 1 个 Agent 已经足够强，为什么还需要 100 个 Agents 来协同？ 51:03 一个构想：Skill 的下一步，可能是「Box」 54:42 无论是 AI 还是人类，高效协作都需要两个前提 59:20 东旭人生的底层驱动力是什么？ 1:01:50 一个野生音乐人是怎么看待 AI 音乐的？ 1:03:24 关于手搓编程的未来畅想1:05:11 彩蛋：一些 AI 浓度很低、存在主义浓度偏高的问题【The gang that made this happen】 制作人：陈皮 剪辑：陈皮 片头：Mondo Bongo - Joe Strummer & The Mescaleros 片尾：Welcome to the Machine - Pink Floyd
