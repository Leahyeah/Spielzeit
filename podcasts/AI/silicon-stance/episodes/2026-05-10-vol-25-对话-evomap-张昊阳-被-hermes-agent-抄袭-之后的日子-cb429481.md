## Text

「膝盖硬，跪不下去。」 过去几个月，张昊阳先后卷入了与硅谷开源项目 OpenClaw 和 Hermes Agent 的「绞杀」与纠纷。面对代码级雷同的质疑和海外联创在 X 上的谩骂，他选择在这一期节目里，把这些问题聊透：关于开源社区的「洋大人」滤镜，关于中国开发者在世界舞台上的创新自信，以及那些「本来不应该讲」的内幕。 但本期节目的含金量远不止于「吃瓜」。你会听到一个关于「Agent 蜂群网络」的激进预言：当「一个 Agent 学会，百万 Agent 继承」，人类的思考速度是否已经成了硅基生命的瓶颈？当编程变成一项软技能，人类最宝贵的价值是否仅剩下那一点点「愿力」与「品味」？ 从 1990 年代的开源精神回望 2026 年的 AI 狂热，这是一场关于降熵、进化的深度对谈。 【主播】硅星人创始人&硅基立场主理人 骆轶航【嘉宾】EvoMap创始人 连续创业者 张昊阳【本期金句】「生命的本质是降熵，进化的本质是无限迭代。」「人最宝贵的价值就在 motivation 和taste，也就是人的直觉。」「 Hermes Agent 的关注度可能只来自于中国。」「没有必要说担心创业公司小团队没有办法定义标准，你只要做的一个更好的东西，那它就是标准。」「 Hermes Agent 的 taste 就明显优于 OpenClaw」「 Evovler 最开始为什么是开源的？因为我当时就压根没想到它能火。」「我们到底是为人做这件事情，还是为 Agent 做？」「你会发现就是 AI 真的非常会走捷径。」
